# Credit-and-Debit-Transaction-

Project Overview
This project presents an interactive financial dashboard built using Microsoft Power BI Desktop to analyze credit and debit transactions across multiple banks, branches, and time periods. It provides insights into transaction behavior, customer activity, and potential risk patterns.

🎯 Project Goals:
Track and compare credit vs debit transactions
Analyze transaction patterns over time
Evaluate bank and branch performance
Detect suspicious and high-risk transactions
Provide data-driven insights for decision-making

📁 Key Features
📊 Real-time interactive dashboard
📈 Monthly & daily transaction trend analysis
🏦 Bank-wise and branch-wise comparisons
🥧 Transaction method distribution (Credit/Debit/Transfer)
⚠️ High-risk transaction identification
